# express-read-app
ionic阅读app后台
=====================
```
一个简单的基于express与mysql的api
```
```
为[ionic-read-app](https://github.com/Taffyw/ionic-read-app)提供了后台服务
```
```
没有使用session储存用户登录信息
```
## 使用这个项目

安装最新版本的 nodejs
修改 \models\db.js mysql数据库连接信息
```bash
$ 本地数据库新建 test1数据库 执行test1.sql文件
```
```bash
$ npm install
```
Then run:

```bash
$ npm run start
```
